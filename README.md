# ActiveX-Auth-UI-Luxury-Dark-Gym-Login-Interface
ActiveX Auth UI is a high-end dark themed authentication interface designed specifically for fitness platforms, gyms, wellness apps and premium lifestyle SaaS products.
It combines luxury black gradients, soft shadows, cinematic typography and an immersive hero visual to create a powerful first impression login experience.

The layout uses a split-screen card system with an interactive motivational slider, modern input components, Google authentication UI, and responsive adaptive layout for all screen sizes.

<b>✨ Key Highlights:</b>

• Luxury black & charcoal gradient design<br>
• Split-screen cinematic auth layout<br>
• Motivational hero slider section<br>
• Soft glassmorphism input fields<br>
• Google login UI integration<br>
• Remember-me & forgot password flow<br>
• Fully responsive desktop & mobile<br>
• High-contrast accessibility friendly<br>
• Clean modern typography<br>
• Perfect for gym, fitness & wellness SaaS<br>
